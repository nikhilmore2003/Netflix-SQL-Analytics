# 📌 Project Overview

This project analyzes Netflix's content catalog using **MySQL**. The analysis combines three datasets containing Netflix content details, production costs, and viewership statistics to uncover meaningful business insights.

The project demonstrates real-world SQL concepts including:

- Data Cleaning
- Joins
- Aggregate Functions
- Window Functions
- Common Table Expressions (CTEs)
- Ranking Functions
- Business Analytics

---

# 🎯 Objectives

- Clean raw Netflix datasets
- Build a relational database
- Analyze Netflix content
- Solve real-world SQL business problems
- Generate business insights
- Present findings using SQL

---

# 📂 Dataset

| Dataset | Description |
|----------|-------------|
| netflix.csv | Netflix content details |
| netflix_content_costs.csv | Production cost, marketing cost and estimated revenue |
| netflix_viewership.csv | Viewership statistics and watch time |
